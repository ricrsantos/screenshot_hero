# PRD-003 - Annotation System

Status: Draft

---

## Objective

Allow users to visually annotate screenshots.

---

## Supported Annotation Types

### Rectangle

### Ellipse

### Arrow

### Line

### Freehand

### Text

### Blur

### Pixelate

### Redaction

### Timestamp

### Number Marker

### Callout

---

## Functional Requirements

### FR-001

User can create annotations.

### FR-002

User can select annotations.

### FR-003

User can move annotations.

### FR-004

User can resize annotations.

### FR-005

User can delete annotations.

### FR-006

User can change color.

### FR-007

User can change stroke width.

### FR-008

Text annotations support editing.

### FR-009

Number markers increment automatically.

### FR-010

Timestamp annotations use configured format.

---

## Acceptance Criteria

✅ Every annotation type can be created

✅ Every annotation type can be edited

✅ Every annotation type can be deleted

✅ Annotation rendering is correct
